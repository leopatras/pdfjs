# This Genero demo uses the pdfjs javascript library

Prerequisites (server side)
  * gnu make 
  * patch
  * node.js and npm
  * bash

Running in GDC
```
  FGLSERVER=<yourgdc> make run
```

Running in GDC via GAS
```
  make gdcwebrun
```
This assumes FGLASDIR is set and the GAS/GDC/fglrun are on the same machine

Just run it in GBC
```
  make webrun
```
